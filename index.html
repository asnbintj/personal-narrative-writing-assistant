가이드 탭
① 대상 선택
대화 상대의 성별과 나이대를 선택하세요. 외국인이라면 하단 외국인 버튼을 누르세요.
② 장소·상황 선택
장소 필터로 좁힌 뒤, 현재 상황에 맞는 항목을 고르세요.
③ 결과 확인
첫 만남 · 재방문 · 관계 형성 탭을 전환하며 단계별 대화 멘트를 확인하세요. 복사 버튼으로 텍스트를 바로 가져갈 수 있어요.
즐겨찾기 저장
결과 화면 우측 상단 🔖 버튼을 눌러 자주 쓰는 상황을 저장해두세요.
둘러보기 탭
장소 · 만남 단계 필터
장소 태그와 만남 단계(첫 만남/재방문/관계형성)를 조합해 전체 상황 목록을 빠르게 탐색할 수 있어요.
항목 클릭
목록에서 항목을 누르면 가이드 탭으로 이동해 해당 상황의 대화 멘트를 바로 볼 수 있어요.
경험담 작성 탭
챗봇 형식으로 작성
봇이 묻는 7가지 질문에 순서대로 답하면 경험담이 자동으로 완성돼요. 건너뛰고 싶은 항목은 "없음"을 입력하세요.
완성 후 복사
작성이 끝나면 복사하기 버튼을 눌러 카카오톡·문자로 담당자에게 보내세요.
